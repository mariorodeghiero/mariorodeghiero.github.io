Gráficos com chartjs

"Construindo gráficos com chart.js"

ChartJs é uma biblioteca de WebGraphics baseado em canvas, podendo realizar manipulação dos dados atraves de javascript e para tornar o desenvolvimento mais rapido e pratico. Com este framework, você ainda gera graficos respovivos e pode personlizar de acordo com sua necessidade, pois ele possui varios tipos diferentes de grafico.

Código HTML

Como este é um simples exemplo para abordar apenas algumas das varias configurações disponiveis no Chartjs, nosso codigo HTML vai apresentar uma simples Tag canvas com id="myChart" e no final, especificamos o caminho do nosso arquivo "chart.js". 


                <html>
                <body>
                
                         <canvas id="myChart"> </canvas>
                
                <script src="js/chart.js"><script>
                </body>
                </html>

Criando o codigo javascript

Primeiramente vamos adicionar um codigo meio que padrão e para dar inicio, vamos criar uma variavel chamada "ctx" para armazenar nesta variavel o elemento que possui o id="myChart". Logo em seguida vamos criar uma variavel chamada myChart e passamos a palavra new com nome Chart para criarmos um novo objeto, passando a variavel "ctx". 


        var ctx = document.getElementById("myChart").getContext('2d');

        var myChart = new Chart(ctx, { 

        });

Dentro do nosso código padrão, existe 3 tipos de configurações que o Chartjs nos proporssiona:

Type

Em type configuramos o tipo de gráfico que desejamos ( line, bar, radar, pie, polarArea, bubble, scatter ) em nosso exemplo vamos usar o tipo de gráfico em linha, mas você pode consultar o site do Chartjs para mais informações. 


        var ctx = document.getElementById("myChart").getContext('2d');

        var myChart = new Chart(ctx, { 
            type: 'line'
        });

Data

Em Data, nos vamos informar os dados para plotar em nosso gráfico. Primeiramente vamos informar os labels, no nosso caso vai ser apresentado alguns meses do ano, que é referenciado por um array, conforme o código abaixo. Na sequência inserimos uma virgula e passamos os datasets para informar os dados a serem inseridos, podendo realizar algumas configurações de estilização, não entrarei em detelhae sobre estilização, pois esta disponivel na documentação, mas no exemplo ja contem uma previa do que é possivel estilizar. 


        var ctx = document.getElementById("myChart").getContext('2d'); 
        var myChart = new Chart(ctx, { 
            type: 'line', 
            data: { 
                    labels: ["Jan", "Fev", "Mar", "Abr", "Mai", "Jun"], 
                        datasets: [{ 
                            label: " Item 1 ", 
                            borderColor: 'rgb(255, 99, 132)', 
                            backgroundColor: 'rgb(255, 99, 132)', 
                            pointHoverBackgroundColor: 'rgb(255, 99, 132)', 
                            fill: false, 
                            data: [0, 10, 5, 2, 20, 30], 
                        }, 
                            { 
                            label: " Item 2 ", 
                            borderColor: 'rgba(0, 207, 214, 1)', 
                            backgroundColor: 'rgba(0, 207, 214, 1)', 
                            pointHoverBackgroundColor: 'rgba(0, 207, 214, 1)', 
                            fill: false, 
                            data: [0, 20, 15, 22, 29, 37], 
                        }, 
                            { 
                            label: " Item 3 ", 
                            borderColor: 'rgba(36, 249, 203, 1)',
                            backgroundColor: 'rgba(36, 249, 203, 1)', 
                            pointHoverBackgroundColor: 'rgba(36, 249, 203, 1)', 
                            fill: false, 
                            data: [0, 10, 25, 32, 19, 27], 
                        }],
                } 
        });

Options

Em options, realizamos as congirações globais do nosso gráfico, então logo após o fechamento da chaves do tipo "data", vamo inserir uma virgula e passar o options. Dentro dele é possivel realizar diverssos tipos de configurações, vou informar e demostrar no código abaixo, apenas algumas delas, mas também esta disponivel na documentação do Chartjs. 



            var ctx = document.getElementById("myChart").getContext('2d');
            var myChart = new Chart(ctx, {
                type: 'line',
                data: {
                    labels: ["Jan", "Fev", "Mar", "Abr", "Mai", "Jun"],
                    datasets: [{
                        label: " Item 1 ",
                        borderColor: 'rgb(255, 99, 132)',
                        backgroundColor: 'rgb(255, 99, 132)',
                        pointHoverBackgroundColor: 'rgb(255, 99, 132)',
                        fill: false,
                        data: [0, 10, 5, 2, 20, 30],
                    },
                                {
                        label: " Item 2 ",
                        borderColor: 'rgba(0, 207, 214, 1)',
                        backgroundColor: 'rgba(0, 207, 214, 1)',
                        pointHoverBackgroundColor: 'rgba(0, 207, 214, 1)',
                        fill: false,
                        data: [0, 20, 15, 22, 29, 37],
                    },
                                {
                        label: " Item 3 ",
                        borderColor: 'rgba(36, 249, 203, 1)',
                        backgroundColor: 'rgba(36, 249, 203, 1)',
                        pointHoverBackgroundColor: 'rgba(36, 249, 203, 1)',
                        fill: false,
                        data: [0, 10, 25, 32, 19, 27],
                    }]
                },
                options: {
                layout: {
                        padding: {
                            left: 50,
                            right: 50,
                            top: 20,
                            bottom: 20
                        }
                        },
                    title: {
                        display: true,
                        text: 'Gráfico Ano 2017',
                        fontSize: 13,
                        padding: 20
                    },
                    legend: {
                        position: 'bottom',
                        labels:{
                            boxWidth: 15
                        },
                    },
                    animation:{
                        easing: 'easeOutBack',
                    },
                    scales: {
                        yAxes: [{
                            gridLines: {
                                display: true,
                                color: "rgba(98, 224, 228, 0.4)"
                            }
                        }],
                        xAxes: [{
                            gridLines: {
                                display: true,
                                color: "rgba(98, 224, 228, 0.4)"
                            }
                        }],
                    }
                }
            });
