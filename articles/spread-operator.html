Spread operator (...)

"Um pouco mais sobre spread operator (...)"

Com spread operator, podemos criar um novo objeto a partir de um objeto existente sem alterar o valor do objeto antigo. Ele é muito utilizado para percorrer um array e realizar alguma ação com os elementos encontrado. Abaixo vamos ver algumas maneiras de como podemos implantar o spread operator, simulando algumas ações. A sintaxe do spread operator é simples, basta usar os três pontos (...).

Exemplo 1

Criaremos um array com três números, logo em seguida temos uma função soma, que recebe três parâmetros e seu retorno é soma de todos parâmetros. Para realizar a soma utilizando o spread operator, simplesmente chamamos a função soma e passamos "..." com o array. 


let arr = [1, 2, 3];

function soma(x, y, z) {
    return x + y + z;
}
soma(...arr); //resultado: 6

Obs:Você podera testar todos exemplos no terminal, usando o Node.js.

Exemplo 2

No código abaixo, queremos obter o maior numero existente em um Array, usando o Spread Operator.
Reparem que temos um Array e em seguida criamos uma variável com nome "maior" que recebe o retorno da função "Math" com a propriedade "max". "Max" é uma propriedade do objeto "Math", que é uma coleção de valores e funções matemáticas. 


let arr2 = [77, 34, 1704, 333, 243];
let maior = Math.max(...arr2);

console.log(maior); //resultado:1704

Exemplo 3

Vamos imaginar que você queira colocar os elementos do array3 dentro do array4, sua primeira tentativa seria o seguinte código abaixo: 


let arr3 = [4, 10, 47];
let arr4 = [21, arr3, 11, 99];

console.log(arr4); // resultado: [21,[4, 10, 47], 11, 99];

O problema no código acima, é que você acaba criando um array dentro de outro array. Analisando o exemplo abaixo, usamos spread operator para realizar a ação. Note!! Que no resultado, os colchetes sumiram de dentro array4, ficando somente elementos numéricos no array. 


let arr3 = [4, 10, 47];
let arr4 = [21, ...arr3, 11, 99];

console.log(arr4); // resultado: [21, 4, 10, 47, 11, 99];

Exemplo 4

Com spread operator, podemos também separar cada letra de uma string, dentro de um array. Veja no exemplo abaixo como é simples realizar esta ação: 


let str = 'code';
let arr = [...str];

console.log(arr); // resultado: ['c', 'o', 'd', 'e']

Exemplo 5

Não podemos esquecer de mostrar que quando usamos o spread operator para criar um novo array com elementos de outro array, toda alteração feita no novo array, não vai ser realizada no antigo array. Vejamos as duas situações abaixo para ficar mais claro: 


/**
* Sem spread operator, onde o antigo array também sofre alteração.
*/

let arr5 = [34, 23, 44];
let arr6 = arr5;

arr6.push(99);

console.log(arr5); // resultado: [34, 23, 44, 99]
        

/**
* Com spread operator, onde o antigo array não sofre alteração.
*/

let arr5 = [34, 23, 44];
let arr6 = [...arr5];

arr6.push(99);

console.log(arr5); // resultado: [34, 23, 44]

Exemplo 6

Com spread operator, podemos criar um objeto com suas chaves e valores. No exemplo abaixo, você pode ver melhor como funciona. Primeiro criamos um array, no exemplo foi array com strings, mas pode ser qualquer tipo. Depois criamos nossa variável obj e passamos a chaves com spread operator(...) com nosso array. Veja o resultado..., você pode acessar qualquer chave do objeto e obter o seu valor. 


var arr = ["mario", "antonio", "rodeghiero"];
var obj = { ...arr };

console.log(obj); // resultado: {'0': 'mario', '1': 'antonio', '2': 'rodeghiero',}
console.log(obj[0]); // resultado: mario
