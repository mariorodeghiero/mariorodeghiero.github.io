styled-components

"css in js com styled-components"

styled-components é uma biblioteca com mais de 2 milhões de downloads que nos possibilita usar css no javascript. O styled-components nos possibilita algumas funcionalidades que nos auxiliam no processo de densenvolvimento como: compatibilidade de browsers, css individual para cada component, não há necessidade de importação de arquivos.css ... você pode conferir mais em styled-components
Agora vamos ver como funcina o styled-components com ReactJS.

Install

Nos exemplos a seguir será usado o "creat-react-app" e "styled-components" por tanto se faz necessario intalar-los para darmos continuidade.

  • creact-react-app

    $ npx create-react-app my-app


  • styled-components

    $ npm install --save styled-components

Exemplo 1

Neste exemplo vamos criar um component chamado "Button.js" e chamamos ele no "App.js" para renderizarmos. Não entrarei nos meritos de como se usa um componente em "react.js", caso você queira aprender mais sobre react.js, tenho alguns artigos que aborda os passos. Nosso componente terá a seguinte estrutura: 

        
    import React, { Component } from 'react';
    import styled from 'styled-components';

    const Button1 = styled.button`
    padding: 10px;
    width: 130px;
    border-radius: 40px;
    top: 50%;
    left: 50%;
    position: absolute;
    margin: 0 auto;
    margin-top: -50px;
    margin-left: -50px;
    :focus {
    box-shadow: 0 0 0 0;
    border: 0 none;
    outline: 0;
    }
    :hover {
    opacity: 0.6;
    }
    `;

    class Home extends Component {
    render() {
    return (
    div>
      Start
); } } export default Home;

Conclusion

These are just a few of the many features that styled-components provides to streamline the development process, leaving our code more performative. It is worth checking the documentation directly on the styled-components and know a little more about the power it offers.
