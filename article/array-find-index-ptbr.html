Array.find( ) e Array.findIndex( )

"Utilizando os metodos Array.find( ) e Array.findIndex( )"

Os métodos Array.find( ) e Array.findIndex( ) são novos métodos introduzidos no ES6, com eles pudemos facilmente verificar se existe determinada "Dado" ou um "Index" dentro de um "Array".

Array.find( )

O método Array.find( ) procura se existe o dado desejado, ele percorre todos elementos de um "Array" até encontrar um valor que retorne "true" e interrompe a execução, caso ele não encontre o dado desejado, sera retornado "undefined". É importante ressaltar que ele não altera o "array original". 

        
          const cars = [
              {
                  model: "HB20",
                  year: "2015"},
              {
                  model: "Onix",
                  year: "2016"
              }
           ]

Supondo que temos dados retornando de um "API" conforme o array de carros acima e queremos realizar uma busca por o carro "Onix".

        
          const cars = [
              {
                  model: "HB20",
                  year: "2015"},
              {
                  model: "Onix",
                  year: "2016"
              }
           ]

          const carFind = cars.find(name => name.model === "Onix")


          console.log(carFind)  //retorno: { model: 'Onix', year: '2016' }

Array.findIndex( )

O método Array.findIndex( ) é bem similar ao "Array.find( )" só que quando ele encontra o primeiro dado da pesquisa realizada, ele retorna a posição do dado no Array.
É importante ressaltar que caso você pesquise por um valor que não existe, sera retornado -1.
No Exemplo abaixo vamos pesquisar por o carro "HB20" que retorna a posição dele no Array e também vamos pesquisar por o carro "Soul" que não existe no Array. 

        
          const cars = [
              {
                  model: "HB20",
                  year: "2015"},
              {
                  model: "Onix",
                  year: "2016"
              }
           ]

          const carIndex = cars.findIndex(name => name.model === "HB20")

          const carNotFound = cars.findIndex(name => name.model === "Soul")


          console.log(carIndex)  //retorno:  0

          console.log(carNotFound)  //retorno:  -1
