Objeto literal

"Construindo objetos de notação literal."

Neste post vamos criar um objeto em javascript, usando notação literal. É um objeto que contem uma coleção de valores referenciados por nomes, e podem ser utilizados conforme a necessidade do seu programa. O objeto pode conter propriedades "dado: valor" e também métodos (quando armazenamos uma função dentro de uma propriedade).

Código HTML

Logo abaixo esta disponível o código HTML que foi utilizando para exemplo. Para estilização foi usado o framework bootstrap e configurado algumas estilizações pessoais, fique a vontade para aplicar suas estilazações.

Construindo um objeto literal

Vamos criar um objeto "Restaurante" no qual vai ter o nome de "Thay", onde vamos ter 30 mesas, sendo que 15 delas estarão reservadas. Como vocês podem ver no código abaixo, primeiramente criamos uma variável "restaurante" seguida do sinal de igual e abrimos chaves para criar o objeto, com suas propriedades e método. Dentro das chaves, vamos inserir cada objeto (nome, mesas, reservada) e por ultimo o método com uma função para checar a quantidade de mesas disponíveis, onde irá retornar o valor da quantidade de mesas, menos a quantidade de mesas reservadas.

Código JS

      
let restaurante = {
    nome: 'Thay',
    mesas: 30,
    reservada: 25,
    mesasDisponiveis: function () {
        return this.mesas - this.reservada;
    }
};

let restNome = document.getElementById('restauranteNome');
restNome.innerHTML = restaurante.nome;

let restMesas = document.getElementById('mesas');
restMesas.textContent = restaurante.mesasDisponiveis();
